##Personal Portfolio Website

##Overview

This project is a responsive personal portfolio website developed as part of Task 1: Environment Setup & Static Website Deployment.
The website showcases my profile, skills, education, projects, and contact information using modern web development practices.

##Features
•	Responsive design for mobile, tablet, and desktop devices
•	Home page with introduction and profile information
•	About page with education, skills, and career goals
•	Contact page with contact details and social links
•	Consistent navigation bar across all pages
•	Modern layout using CSS Flexbox and Grid
•	Clean and user-friendly interface

##Technologies Used
•	HTML5
•	CSS3
•	CSS Flexbox
•	CSS Grid
•	Git
•	GitHub
•	GitHub Pages / Netlify

Setup Instructions
Clone the Repository
git clone https://github.com/mrballipati07-pixel/portfolio-site.git
Open the Project
Navigate to the project folder:
cd portfolio-site
Run the Website
Open the index.html file in your browser
OR
Use VS Code Live Server:
1.	Open the project in VS Code
2.	Install the Live Server extension
3.	Right-click on index.html
4.	Select Open with Live Server
